vanilla-webdev-vagrant
======================

Webdevelopers Base Vagrant box

# Getting Startet

1. Clone the project. 
2. Run the command `vagrant up`
3. redirect your Browser to http://localhost:8080

# phpMyAdmin

http://localhost:8080/phpmyadmin/

# What's Installed

+ Ubuntu Precise (12.04)
+ Wordpress 3.8
+ Mysql
+ Php
+ Phpmyadmin
+ Subversion
+ PEAR
+ Xdebug
+ PHPUnit
+ phploc
+ phpcpd
+ phpdcd
+ phpcs
+ phpdepend
+ phpmd
+ PHP_CodeBrowser
